# 每日站立會議 (Daily Standup)

## 會議規則

| 項目 | 說明 |
|------|------|
| 時間 | 每日 09:30 ~ 09:45 |
| 地點 | 會議室 / 線上 |
| 參與者 | 開發團隊 |
| 發言限時 | 每人 2 分鐘 |

---

## 發言模板

### 每位成員需回答：

1. **昨天做了什麼？**
   - 完成的任務
   - 交付的成果

2. **今天要做什麼？**
   - 預計完成的任務
   - 目標產出

3. **有什麼阻礙？**
   - 遇到的問題
   - 需要的幫助
