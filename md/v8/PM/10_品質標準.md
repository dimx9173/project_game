# 品質標準 (Quality Standards)

## 程式碼品質

### 程式碼規範

| 項目 | 標準 |
|------|------|
| 語言 | TypeScript |
| Linting | ESLint |
| 格式化 | Prettier |
| 命名 | camelCase / PascalCase |
| 註解 | JSDoc 必要時添加 |

### 程式碼審查 (Code Review)

- 所有 PR 必須經過 1 人以上 Review
- 重大變更需 2 人以上 Review
- Review 必須在 24 小時內完成
- 建議修改必須處理
