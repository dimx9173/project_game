## 2.1 玩家註冊與登入流程

```text
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   玩家      │     │  單機本地   │     │  集中式    │
│  (玩家端)   │     │   後台     │     │   後台     │
└──────┬──────┘     └──────┬──────┘     └──────┬──────┘
       │                   │                   │
       │  1. 註冊請求      │                   │
       │─────────────────>│                   │
       │                   │                   │
       │  2. 驗證手機/驗證碼                   │
       │<─────────────────│                   │
       │                   │                   │
       │  3. 填寫基本資料  │                   │
       │─────────────────>│                   │
       │                   │                   │
       │  4. 建立本地帳號  │                   │
       │                   │                   │
       │  5. [離線模式] 直接登入               │
       │<─────────────────│                   │
       │                   │                   │
       │  6. [連線模式] 同步到中央             │
       │                   │─────────────────>│
       │                   │                   │
       │                   │  7. 建立中央帳號  │
       │                   │<─────────────────│
       │                   │                   │
       │  8. 登入成功      │                   │
       │<─────────────────│                   │
       │                   │                   │
```
