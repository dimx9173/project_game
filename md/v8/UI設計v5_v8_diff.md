# 遊戲平台管理系統：UI 設計 V5 與 V8 版本差異分析

## 1. 文件概述
本文件旨在記錄由「V5.0 整合版 (Corporate Precision)」升級至「V8.0 旗艦版 (Premium Dark + Cyber Purple)」的重大變更，作為開發團隊與 UI 設計同仁的對照參考。

---

## 2. 核心視覺系統差異

| 項目 | V5.0 整合版 (2026-02-23) | V8.0 旗艦版 (2026-03-02) |
| :--- | :--- | :--- |
| **設計風格** | Corporate Precision (商務專業) | Premium Dark + Glassmorphism (旗艦科技) |
| **主背景色** | #FFFFFF (純白) | #0a0a0f (深邃黑) |
| **主題色系** | #4F46E5 (靛藍) | #7c3aed (霓虹紫) |
| **視覺效果** | 扁平化、清晰邊界 | 玻璃擬態、霓虹發光、背景虛化 (Blur) |
| **字體系統** | DM Sans / Noto Sans TC | Noto Sans TC / JetBrains Mono (強調等寬數據) |
| **技術堆疊** | React / Tailwind / shadcn | Blazor Server / Radzen Blazor |

---

## 3. 功能模組演進

### 3.1 V8 版本新增/強化之功能
*   **OTA 遠端版本派送 (強化)**：V8 明確定義了檔案上傳、版號賦予、機台自動偵測、MD5 完整性校驗等完整流程。
*   **進階硬體監控中心 (新增)**：整合 CPU、記憶體佔用、網路延遲 (Heartbeat) 的即時儀表板，具備監測告警功能。
*   **對帳與爭議帳單管理 (新增)**：具備 Audit 審計標記、沖銷憑證產生邏輯。遵循會計原則：不改原始資料，僅產生沖銷紀錄。
*   **代理商樹狀結構管理 (新增)**：支持代理商三層架構（Admin > Agent > Operator），具備層級權限繼承與自動分潤計算。

### 3.2 V5 版本中有、但 V8 版本未列出之功能 (注意點)
*   **介面設定 (Interface Settings)**：V5 含有 `/ui-config` 路由，可用於配置「玩家端大廳」的視覺佈局，V8 暫未明確列出。
*   **遊戲測試 - 虛擬下注 (Game Test)**：V5 的單機後台包含一個「測試模式」，允許管理員在現場機台進行虛擬下注以驗證邏輯。
*   **物理硬體調節**：V5 明確列出了「音量」與「亮度」調整區塊，V8 傾向於抽象化的系統設定。

---

## 4. 交互設計規範變更

*   **動畫系統**：V8 引入了動態背景（8s 緩慢浮動）、脈衝點（Pulse Dot）代表即時連線狀態、以及按鈕點擊的收縮 (0.95 Scale) 反饋。
*   **數據呈現**：V8 全面採用 `JetBrains Mono` 字體處理所有金額、流水號與機台代碼，確保在複雜的後台列表中具備極高的一致性與可讀性。
*   **操作回饋**：V8 的卡片設計採用 `Glow` 效果，當滑鼠懸停 (Hover) 時會產生外發光陰影。

---

## 5. 總結
V8 版本標誌著系統從「通用後台工具」進化為「產業深度定製系統」。視覺上的深色科技感更符合遊戲產業調性，功能上也由單純的「資料紀錄」進化為「營運自動化與安全稽核」。
