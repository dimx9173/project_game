## 3.5 介面設定操作

### [C77] 遊戲分類排序

**操作流程**：

```
1. 進入「分類排序」
2. 拖曳調整順序
3. 設定顯示/隱藏
4. 儲存排序
```

**功能**：

- 拖曳排序
- 隱藏/顯示分類
- 設定分類圖示

**畫面 Mockup**：
![遊戲分類](./mockup/central/mockup_std_game_category.png)

### [C80] 推薦遊戲設定

**操作流程**：

```
1. 進入「推薦遊戲」
2. 選擇要推薦的遊戲
3. 設定推薦位置
4. 設定顯示期間
5. 儲存設定
```

**推薦類型**：

| 類型     | 說明         |
| -------- | ------------ |
| 首頁推薦 | 大廳首頁顯示 |
| 熱門推薦 | 熱門遊戲區塊 |
| 新遊戲   | 最新上架遊戲 |
| 活動推薦 | 活動相關遊戲 |

### [C76] 公告管理

**操作流程**：

```
1. 進入「公告管理」
2. 新增/編輯公告
3. 設定發布時間
4. 選擇顯示位置
5. 儲存公告
```

**公告欄位**：

| 欄位名稱 | 類型     | 說明                  |
| -------- | -------- | --------------------- |
| 標題     | 文字     | 公告標題              |
| 內容     | 文字     | 公告內容（支援 HTML） |
| 類型     | 下拉選單 | 系統/活動/維護        |
| 顯示位置 | 多選     | 大廳/遊戲內/登入頁    |
| 開始時間 | 日期時間 | 顯示開始時間          |
| 結束時間 | 日期時間 | 顯示結束時間          |
| 狀態     | 開關     | 啟用/停用             |

### [C81] 前端版面廣播模組

**功能說明**：

即時廣播訊息到所有在線客戶端，包含系統公告、遊戲通知、跑馬燈、彈出訊息等。支援單次發送、排程發送、目標篩選及效果追蹤。

**操作流程**：

```
1. 進入「廣播管理」
2. 選擇廣播類型
3. 編輯廣播內容
4. 選擇目標範圍
5. 設定發送時間（即時/排程）
6. 發送廣播
7. 追蹤廣播效果
```

#### 3.5.4.1 廣播類型說明

| 類型     | 代碼      | 顯示位置 | 顯示方式 | 適用場景           |
| -------- | --------- | -------- | -------- | ------------------ |
| 跑馬燈   | marquee   | 畫面頂部 | 滾動文字 | 重要公告、優惠活動 |
| 彈出視窗 | popup     | 螢幕中央 | 強制顯示 | 系統通知、活動提醒 |
| 系統訊息 | system    | 通知區域 | 一般訊息 | 一般性通知         |
| 遊戲通知 | game      | 遊戲內   | 遊戲邊緣 | 遊戲相關資訊       |
| 全域廣播 | broadcast | 所有頁面 | 頂部橫幅 | 緊急公告、維護通知 |

**內容格式支援**：

- 純文字
- 文字 + 表情符號
- 文字 + 圖片（URL）
- 文字 + 超連結

#### 3.5.4.2 目標篩選條件

| 站點範圍 | 多選     | 選擇目標站點     |
| 機台等級 | 多選     | 機台分組篩選     |

### [C78] 前端大廳配置 (/ui-config)

**功能說明**：
提供管理大廳前端 UI 佈景（與顯示比例）的預設功能，包含主題顏色、背景圖示及風格選項。機台重新載入後即可自動從此端點抓取最新配置。
| 建立時間 | 日期區間 | 限定機台新增時間 |
| 餘額範圍 | 數值區間 | 限定機台餘額範圍 |
| 機台類型 | 多選     | 限定特定機台     |
| 遊戲類型 | 多選     | 限定特定遊戲     |

**預設目標選項**：

| 選項     | 說明                 |
| -------- | -------------------- |
| 全體機台 | 發送給所有機台       |
| 在線機台 | 僅發送給當前在線機台 |
| 指定機台 | 選擇特定機台發送     |
| 指定站點 | 選擇特定站點發送     |

#### 3.5.4.3 排程廣播

**排程類型**：

| 類型     | 說明               | 範例              |
| -------- | ------------------ | ----------------- |
| 立即發送 | 立即推送到客戶端   | 緊急公告          |
| 延遲發送 | 設定延遲時間後發送 | 5分鐘後提醒       |
| 定時發送 | 指定精確時間發送   | 每日下午3點       |
| 循環發送 | 設定循環週期發送   | 每小時提醒        |
| 條件觸發 | 滿足條件自動發送   | 餘額低於100元提醒 |

**排程參數**：

| 參數     | 類型     | 說明                 |
| -------- | -------- | -------------------- |
| 開始時間 | 日期時間 | 排程開始時間         |
| 結束時間 | 日期時間 | 排程結束時間         |
| 發送間隔 | 數字     | 循環發送間隔（分鐘） |
| 發送次數 | 數字     | 循環發送次數上限     |
| 時區     | 下拉選單 | 時區設定             |

#### 3.5.4.4 廣播效果追蹤

**追蹤指標**：

| 指標   | 說明               | 計算方式              |
| ------ | ------------------ | --------------------- |
| 送達率 | 訊息成功送達比例   | 成功送達數 / 發送總數 |
| 閱讀率 | 機台端閱讀訊息比例 | 閱讀數 / 送達數       |
| 點擊率 | 機台端點擊連結比例 | 點擊數 / 閱讀數       |
| 忽略率 | 機台端未閱讀比例   | 未閱讀數 / 送達數     |

**效果報表欄位**：

| 廣播編號 | 文字     | 系統自動產生 |
| 廣播標題 | 文字     | 廣播訊息標題 |
| 類型     | 標籤     | 廣播類型     |
| 發送時間 | 日期時間 | 實際發送時間 |
| 目標數   | 數字     | 目標機台數   |
| 送達數   | 數字     | 成功送達數   |
| 閱讀數   | 數字     | 已閱讀數     |
| 點擊數   | 數字     | 點擊連結數   |

**匯出功能**：

- 支援匯出效果報表為 CSV 或 Excel
- 支援篩選特定時間範圍

#### 3.5.4.5 輸入欄位總覽

| 廣播標題 | 文字     | ✅   | 廣播訊息標題          |
| 廣播類型 | 下拉選單 | ✅   | 選擇廣播類型          |
| 廣播內容 | 文字     | ✅   | 訊息內容（支援 HTML） |
| 目標範圍 | 多選     | ✅   | 選擇目標機台          |
| 篩選條件 | 複合     | -    | 進階篩選條件          |
| 發送時間 | 日期時間 | ✅   | 即時/排程選擇         |
| 顯示時長 | 數字     | -    | 訊息顯示秒數          |
| 排序權重 | 數字     | -    | 多則訊息排序優先級    |
