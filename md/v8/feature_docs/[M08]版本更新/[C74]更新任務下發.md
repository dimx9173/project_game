### [C74] 發布版本

**操作流程**：

```
1. 選擇要發布的版本
2. 選擇測試機台 (2-3台)
3. 開始測試派發
4. 監控 24 小時
5. 確認無異常
6. 正式派發 (全部/分批)
```

**派發模式**：

| 模式     | 說明               |
| -------- | ------------------ |
| 測試派發 | 僅派發至指定機台   |
| 全部派發 | 一次派發至所有機台 |
| 分批派發 | 按批次逐步派發     |
| 定時派發 | 設定時間自動派發   |
