### [C62] 牌靴管理

**功能說明**：
管理牌靴（ Shoe ）的副數、張數、換靴規則。

**操作流程**：

```
1. 選擇牌桌
2. 進入「牌靴管理」
3. 設定牌靴參數
4. 查看當前牌靴狀態
```

**設定欄位**：

| 欄位名稱           | 類型     | 說明                       |
| ------------------ | -------- | -------------------------- |
| 牌靴副數           | 數字     | 1-8 副牌                   |
| 剩餘多少張觸發換靴 | 數字     | 建議 20-40 張              |
| 換靴規則           | 下拉選單 | 自動/手動                  |
| 備註               | 文字     | 依照第三方遊戲商的API 調整 |
