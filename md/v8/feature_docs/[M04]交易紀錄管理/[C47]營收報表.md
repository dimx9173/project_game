### [C47] 營收報表

**操作流程**：

```plaintext
1. 進入「報表」模組
2. 選擇「營收報表」
3. 設定時間範圍
4. 選擇篩選條件（可多選）
5. 產生報表
6. 匯出或列印
```

**篩選維度**（且可多選）：

| 維度             | 說明                       | 範例                            |
| ---------------- | -------------------------- | ------------------------------- |
| **門市**   | 按分店/門市篩選（可多選）  | 台北門市A區、台中門市、高雄門市 |
| **機台**   | 按機台篩選（可多選）       | 機台編號、機台名稱              |
| **遊戲**   | 按遊戲篩選（可多選）       | 老虎機、捕魚機、棋牌            |
| **遊戲商** | 按遊戲供應商篩選（可多選） | RG電子、JDB電子、RT電子         |

**報表類型**：

| 類型   | 說明         |
| ------ | ------------ |
| 日報表 | 每日營收統計 |
| 週報表 | 每週營收統計 |
| 月報表 | 每月營收統計 |
| 年報表 | 每年營收統計 |

**營收欄位**：

| 欄位      | 說明             |
| --------- | ---------------- |
| 總營收    | 派彩 - 下注      |
| 下注總額  | 所有下注金額合計 |
| 派彩總額  | 所有派彩金額合計 |
| 遊戲局數  | 總遊戲次數       |
| 營收(RTP) | 派彩率           |
