## 3.6 機台管理操作

### [C101] 新增機台

**操作流程**：

```
1. 點擊「新增機台」按鈕
2. 填寫機台基本資料
3. 設定機台參數
4. 驗證機台連線
5. 完成新增
```

**輸入欄位**：

| 欄位名稱 | 類型     | 必填 | 說明                       |
| -------- | -------- | ---- | -------------------------- |
| 機台編號 | 文字     | ✅   | 唯一識別碼，格式：MCH-XXXX |
| 機台名稱 | 文字     | ✅   | 自訂名稱，最長 50 字元     |
| 機台型號 | 下拉選單 | ✅   | 選擇硬體型號               |
| 所屬站點 | 下拉選單 | ✅   | 選擇所屬營業站點           |
| IP 位址  | IP       | ✅   | 機台網路位置               |
| 連接埠   | 數字     | ✅   | 預設 8080                  |
| 遊戲類型 | 多選     | ✅   | 老虎機/捕魚/棋牌           |
| 備註     | 文字     | -    | 選填備註說明               |

### [C101] 編輯機台

**操作流程**：

```
1. 點擊機台列表中的編輯按鈕
2. 修改機台資料
3. 儲存變更
4. 系統記錄操作日誌
```

**可編輯欄位**：

| 欄位名稱 | 說明           |
| -------- | -------------- |
| 機台名稱 | 修改顯示名稱   |
| 所屬站點 | 變更所屬站點   |
| IP 位址  | 更新網路設定   |
| 機台狀態 | 線上/離線/維護 |
| 備註     | 更新備註資訊   |

---

### [C101] 刪除機台

**操作流程**：

```
1. 選取要刪除的機台
2. 點擊刪除按鈕
3. 確認刪除對話框
4. 執行軟刪除（保留資料）
```

**刪除規則**：

- 僅可刪除離線狀態機台
- 刪除前需確認無進行中交易
- 系統保留刪除紀錄 90 天

### [C12] 遠端指令

**支援指令**：

| 指令名稱 | 代碼       | 說明             |
| -------- | ---------- | ---------------- |
| 重新啟動 | restart    | 遠端重啟機台系統 |
| 關機     | shutdown   | 遠端關閉機台     |
| 開分     | cash_in    | 設定機台開分分數 |
| 洗分     | cash_out   | 設定機台洗分分數 |
| 強制同步 | force_sync | 立即同步資料     |

### [L05] 機台列表檢視
**功能說明**：顯示所有機台的基本資訊，支援本機快取以確保離線檢視。

### [L06] 機台狀態監控
**功能說明**：即時或定時回報機台的連線狀況與硬體狀態。

### [C107] 機台標籤管理
**功能說明**：允許管理者為機台自訂標籤，方便進行分組查詢與多標籤篩選。

### [C11] 開分/洗分
**功能說明**：遠端發送指令為機台增加或減少玩家點數，需配合嚴格的操作紀錄審計。

### [L07] 機台配置下發
**功能說明**：從集中式後台統一寄送組態檔案至特定機台，機台接收後生效。

### [C14] 機台設定
**功能說明**：針對遠端機台的網路參數、系統時間等進行基礎覆寫設定。

### [C15] 測試模式
**功能說明**：切換機台至測試環境，允許進行不扣除實際餘額的測試遊玩。
