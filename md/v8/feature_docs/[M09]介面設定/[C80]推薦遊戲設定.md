### [C80] 推薦遊戲設定

**操作流程**：

```
1. 進入「推薦遊戲」
2. 選擇要推薦的遊戲
3. 設定推薦位置
4. 設定顯示期間
5. 儲存設定
```

**推薦類型**：

| 類型     | 說明         |
| -------- | ------------ |
| 首頁推薦 | 大廳首頁顯示 |
| 熱門推薦 | 熱門遊戲區塊 |
| 新遊戲   | 最新上架遊戲 |
| 活動推薦 | 活動相關遊戲 |
