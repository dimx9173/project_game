### [C81] 前端版面廣播模組

**功能說明**：

即時廣播訊息到所有在線客戶端，包含系統公告、遊戲通知、跑馬燈、彈出訊息等。支援單次發送、排程發送、目標篩選及效果追蹤。

**操作流程**：

```
1. 進入「廣播管理」
2. 選擇廣播類型
3. 編輯廣播內容
4. 選擇目標範圍
5. 設定發送時間（即時/排程）
6. 發送廣播
7. 追蹤廣播效果
```

#### 3.5.4.1 廣播類型說明

| 類型     | 代碼      | 顯示位置 | 顯示方式 | 適用場景           |
| -------- | --------- | -------- | -------- | ------------------ |
| 跑馬燈   | marquee   | 畫面頂部 | 滾動文字 | 重要公告、優惠活動 |
| 彈出視窗 | popup     | 螢幕中央 | 強制顯示 | 系統通知、活動提醒 |
| 系統訊息 | system    | 通知區域 | 一般訊息 | 一般性通知         |
| 遊戲通知 | game      | 遊戲內   | 遊戲邊緣 | 遊戲相關資訊       |
| 全域廣播 | broadcast | 所有頁面 | 頂部橫幅 | 緊急公告、維護通知 |

**內容格式支援**：

- 純文字
- 文字 + 表情符號
- 文字 + 圖片（URL）
- 文字 + 超連結

#### 3.5.4.2 目標篩選條件

| 站點範圍 | 多選     | 選擇目標站點     |
| 機台等級 | 多選     | 機台分組篩選     |
