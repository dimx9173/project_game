### [L11] 遊戲列表檢視

**操作流程**：

```plaintext
1. 進入「遊戲管理」模組
2. 檢視遊戲列表
3. 使用篩選條件搜尋遊戲
4. 點擊遊戲查看詳情
```

**輸入欄位**：

| 欄位名稱 | 類型     | 說明                  |
| -------- | -------- | --------------------- |
| 遊戲編號 | 文字     | 系統自動產生          |
| 遊戲名稱 | 文字     | 遊戲顯示名稱          |
| 遊戲類型 | 下拉選單 | 老虎機/捕魚/棋牌/街機 |
| 供應商   | 下拉選單 | 遊戲所屬供應商        |
| 狀態     | 標籤     | 啟用/停用/維護中      |
| RTP 率   | 數值     | 機台回報率百分比      |
