### [L10] 遊戲版本管理

**操作流程**：

```
1. 選擇遊戲後點擊「版本管理」
2. 查看版本歷史
3. 上傳新版本
4. 設定版本為正式版
```

**版本欄位**：

| 欄位名稱 | 類型 | 說明                    |
| -------- | ---- | ----------------------- |
| 版本號   | 文字 | 格式：Major.Minor.Patch |
| 上架日期 | 日期 | 版本發布日期            |
| 狀態     | 標籤 | 測試中/正式版/已下架    |
| 更新內容 | 文字 | 版本變更說明            |
