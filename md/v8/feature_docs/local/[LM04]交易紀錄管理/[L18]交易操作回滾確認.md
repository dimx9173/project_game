### [L18] 交易沖銷

**操作流程**：

```
1. 選取要沖銷的交易
2. 點擊「沖銷」按鈕
3. 填寫沖銷原因
4. 系統驗證沖銷資格
5. 執行沖銷
6. 產生反向交易
7. 記錄沖銷人員
```

**沖銷規則**：

- 僅可沖銷 24 小時內交易
- 需主管授權
- 自動產生沖銷憑證（PDF）

**輸入欄位**：

| 欄位名稱 | 類型     | 必填 | 說明         |
| -------- | -------- | ---- | ------------ |
| 沖銷原因 | 下拉選單 | ✅   | 選擇沖銷原因 |
| 詳細說明 | 文字     | ✅   | 補充說明     |
| 授權碼   | 數字     | ✅   | 主管授權碼   |
