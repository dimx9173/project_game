# [L34] 同步狀態檢視

**功能代碼**: L34  
**所屬模組**: [M05]系統  
**最後更新**: 2026-03-09

---

## 1. 功能說明

同步狀態檢視功能提供管理者一個即時的儀表板，用以監控機台與中央後台之間的資料交換情形。透過此介面，管理者可以確認網路連線品質、資料是否即時上傳以及是否存在待處理任務。

### 主要功能：
- **連線狀態監控**：顯示與中央伺服器的連線狀態（WebSocket/REST）。
- **同步統計**：顯示最近同步時間、待上傳交易筆數。
- **異常警示**：當同步中斷或佇列堆積時發出視覺提醒。

---

## 2. 操作流程

### 2.1 查看同步狀態
1. **進入系統儀表板**：登入本地後台首頁或進入「系統管理」->「同步設定」。
2. **檢視狀態指標**：
   - **雲端連線**：綠色表示在線，紅色表示離線。
   - **最後同步**：顯示最後一次成功完成雙向同步的時間點。
3. **檢查待上傳項**：若「待同步交易」不為 0，表示系統正處於非同步模式或網路連線不穩定，正在快取資料。

### 2.2 手動觸發即時同步
1. **連線檢查**：確認機台處於「在線」狀態。
2. **執行同步**：點擊「立即同步」按鈕。
3. **進度監控**：觀察進度條，查看交易記錄與設定更新的下載進度。
4. **結果確認**：完成後檢視「同步完成」提示及更新後的同步時間。

## 3. 屬性/技術資訊

| 屬性名稱         | 說明                                 | 類型     | 取值範例             |
| ---------------- | ------------------------------------ | -------- | -------------------- |
| 連線狀態 (Link)  | 與中央伺服器的實體/邏輯連線          | Status   | ONLINE / OFFLINE     |
| 同步健康度 (Health)| 資料同步的延遲與成功率             | Level    | 優 / 良 / 差         |
| 待同步數量       | 目前存儲在 `sync_queue` 中待上傳的項 | Number   | 0 - 10,000           |
| 最後同步時間     | 成功的完整同步時間戳                 | DateTime | 2026-03-09 10:00     |
| 網路延遲         | 往返中央伺服器的毫秒數               | Number   | 15ms                 |
