## 📋 概述

本地機台儀表板顯示營收趨勢圖表，提供視覺化的營收數據分析。

---

## 🔄 功能說明

### 主要功能
- 顯示今日營收趨勢（每小時）
- 顯示本週營收趨勢（每日）
- 顯示本月營收趨勢（每日）
- 支援圖表切換（折線圖/柱狀圖）

### 數據來源
- 本地快取（來自集中式後台同步）
- 離線時顯示最後同步的數據

## 🔌 API 調用

### 數據 API

| API | 方法 | 說明 |
|------|------|------|
| `/api/local/dashboard/revenue-trend` | GET | 獲取營收趨勢數據（快取） |

### 數據格式

```json
{
  "today": [
    {"hour": "00:00", "revenue": 10000},
    {"hour": "01:00", "revenue": 15000}
  ],
  "week": [
    {"date": "2026-03-01", "revenue": 500000},
    {"date": "2026-03-02", "revenue": 600000}
  ],
  "month": [
    {"date": "2026-03-01", "revenue": 500000},
    {"date": "2026-03-02", "revenue": 600000}
  ]
}
```
