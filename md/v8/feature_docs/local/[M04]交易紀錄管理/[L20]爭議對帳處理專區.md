### [L20] 歷史記錄

**歷史記錄功能**：

- 完整保留所有爭議單處理歷程
- 可追蹤每個狀態的變更時間
- 記錄所有操作人員
- 保留相關通聯記錄

**歷史記錄欄位**：

| 欄位       | 說明           |
| ---------- | -------------- |
| 爭議單編號 | 爭議單唯一識別 |
| 機台編號   | 發生問題的機台 |
| 爭議類型   | 問題類型       |
| 建立時間   | 爭議單建立時間 |
| 處理時間   | 總處理時長     |
| 裁決結果   | 最終裁決       |
| 處理人     | 最後處理人員   |
| 結案時間   | 結案時間       |

**篩選與查詢**：

| 篩選條件 | 類型       | 說明         |
| -------- | ---------- | ------------ |
| 日期區間 | 日期picker | 建立日期範圍 |
| 爭議類型 | 多選       | 問題類型篩選 |
| 狀態     | 多選       | 爭議單狀態   |
| 緊急程度 | 多選       | 緊急程度     |
| 處理人   | 多選       | 負責人員     |
| 機台編號 | 文字       | 機台編號搜尋 |

**統計報表**：

| 指標         | 說明                   |
| ------------ | ---------------------- |
| 爭議總數     | 期間內建立的所有爭議單 |
| 爭議成立率   | 成立數 / 總數          |
| 平均處理時間 | 從建立到結案平均天數   |
| 熱門爭議類型 | 各類型數量排行         |
| 處理人績效   | 各處理人處理數量與時效 |

**匯出功能**：

- 支援匯出歷史記錄為 CSV 或 Excel
- 支援自訂欄位匯出
