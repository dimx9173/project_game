### [L02] 機台狀態

**功能說明**：
顯示本機硬體狀態（CPU、記憶體、硬碟）、網路狀態（延遲、IP）、遊戲狀態、系統資訊等。

**監控項目**：

| 項目     | 說明                  |
| -------- | --------------------- |
| 硬體狀態 | CPU/記憶體/硬碟使用率 |
| 網路狀態 | 延遲、IP 位址         |
| 遊戲狀態 | 目前遊戲、版本        |
| 系統資訊 | 運行時間、同步狀態    |
