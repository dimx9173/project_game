# 本地機台註冊與登入流程說明

## 📋 概述

本文檔說明單機本地後台的簡化註冊與登入機制。

---

## 🔄 首次開機（註冊）

```
┌─────────────────────────────────────────────────────────────┐
│  機台首次開機                                                │
│     ↓                                                       │
│  顯示「登入」畫面                                            │
│     ↓                                                       │
│  輸入預設 PIN：00000000                                      │
│     ↓                                                       │
│  進入本地設定頁面                                            │
│     ↓                                                       │
│  完成機台初始化設定（機台名稱、連線設定等）                    │
│     ↓                                                       │
│  設定本地 PIN 碼（取代預設 00000000）                         │
│     ↓                                                       │
│  註冊完成 → 機台自動向中央伺服器註冊（若連線）                 │
└─────────────────────────────────────────────────────────────┘
```

### 首次開機流程說明

| 步驟 | 操作 | 說明 |
|------|------|------|
| 1 | 輸入 PIN 00000000 | 預設密碼，僅限首次使用 |
| 2 | 進入本地設定 | 直接進入設定頁面，無需額外驗證 |
| 3 | 設定機台資訊 | 機台名稱、中央伺服器位址等 |
| 4 | 設定本地 PIN | 設定 8 位數字 PIN，之後登入使用 |
| 5 | 自動註冊 | 若連線，自動向中央註冊該機台 |

```
│                    本地後台登入                              │

│              │                             │                │

│              │                             │                │
│              └─────────────────────────────┘                │
│                                                             │
│                      [ 登入 ]                               │
│                                                             │

│           連線狀態: ● 已連線 / ○ 離線                        │

```

### 登入驗證邏輯

```
輸入 PIN 碼
    ↓
┌────────────────────────────────────────────────────────────┐

└────────────────────────────────────────────────────────────┘
    ↓                           ↓
   否                           是
    ↓                           ↓

    ↓                           ↓
   是                      正確？
    ↓                           ↓
進入首次設定              是    否
                               ↓    ↓
                          登入成功  錯誤提示
```

## 📝 PIN 碼規則

| 類型 | 預設 PIN | 自訂 PIN |
|------|---------|---------|
| **數值** | 00000000 | 8 位數字 |
| **使用時機** | 僅限首次開機 | 日常登入使用 |
| **變更方式** | 首次設定時強制變更 | 本機設定頁面變更 |

### PIN 碼安全機制

- **錯誤 5 次鎖定**：連線後可由中央重設，或等待 30 分鐘自動解鎖
- **首次強制變更**：輸入 00000000 後必須設定新 PIN
- **離線可用**：登入驗證完全本地進行，不依賴網路

## 🎯 使用場景範例

### 場景一：新機台首次開機

1. 駐點工程師收到新機台，首次開機
2. 顯示登入畫面，提示「首次開機請輸入 00000000」
3. 輸入 00000000，進入本地設定頁面
4. 設定機台名稱「旗艦店 A-01」
5. 設定中央伺服器位址
6. 設定本地 PIN 碼：12345678
7. 設定完成，機台自動向中央註冊
8. 之後登入輸入 12345678 即可

### 場景二：日常登入（連線狀態）

1. 機台正常連線運作中
2. 顯示登入畫面
3. 輸入本地 PIN：12345678
4. 本地驗證成功，進入本地後台
5. 可執行所有本地功能
6. 若需執行中央授權操作，再使用中央帳號登入

### 場景三：離線緊急處理

1. 機台網路異常，顯示「離線」狀態
2. 輸入本地 PIN：12345678
3. 本地驗證成功（不需連線）
4. 進入本地後台，調整網路設定
5. 網路恢復後自動同步

### 場景四：忘記 PIN

1. 無法登入，聯繫管理員
2. 管理員在集中式後台「機台管理」中找到該機台
3. 執行「重設本地 PIN」功能
4. 產生一次性重設碼
5. 駐點工程師在登入畫面輸入重設碼
6. 強制進入設定頁面，設定新 PIN

## ⚠️ 注意事項

1. **首次開機必須連線**：雖然可用 00000000 登入，但建議首次設定時連線，以便自動向中央註冊
2. **00000000 僅限首次**：設定新 PIN 後，00000000 自動失效
3. **PIN 碼遺失處理**：需透過中央後台重設，無法自行破解
4. **離線功能限制**：僅能調整本地設定，無法執行需中央授權的操作
