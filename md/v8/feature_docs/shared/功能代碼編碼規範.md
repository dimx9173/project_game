# 功能代碼編碼規範與功能矩陣對應說明

## 📋 概述

本文檔定義功能代碼（Function Code）的編碼規則，以及功能文件與功能矩陣的對應方式。

**現有功能矩陣編號**: [Cxx] 集中式功能 / [Lxx] 本地功能 / [Dxx] 部署功能

**文件結構**: 功能說明文件已按 Central/Local 分離存放

---

## 📁 文件結構

### Central（集中式後台功能）

路徑：`feature_docs/central/[CMxx] 模組名稱/[Cxx] 功能名稱.md`

**適用範圍**: 所有 [Cxx] 開頭的功能代碼

範例：
```
feature_docs/central/[CM01] 儀表板/[C01]KPI 監控.md
feature_docs/central/[CM02] 機台管理/[C101] 機台新增編輯刪除.md
feature_docs/central/[CM02] 機台管理/[C11] 開分洗分.md
feature_docs/central/[CM03] 遊戲管理/[C16] 遊戲新增編輯刪除.md
feature_docs/central/[CM12] 對帳同步/[C93] 手動同步.md
```

### Local（本地後台功能）

路徑：`feature_docs/local/[LMxx] 模組名稱/[Lxx] 功能名稱.md`

**適用範圍**: 所有 [Lxx] 開頭的功能代碼

範例：
```
feature_docs/local/[LM01] 儀表板/[L01] 營收趨勢圖.md
feature_docs/local/[LM01] 儀表板/[L02] 機台狀態圖.md
feature_docs/local/[LM02] 機台管理/[L05] 機台列表檢視.md
feature_docs/local/[LM02] 機台管理/[L06] 機台狀態監控.md
feature_docs/local/[LM03] 遊戲管理/[L11] 遊戲列表檢視.md
feature_docs/local/[LM11] 系統設定/[L51] 本地機台註冊與登入流程.md
feature_docs/local/[LM11] 系統設定/[L52] 本地機台雙軌制 PIN 登入流程.md
```

### Shared（共用文件）

路徑：`feature_docs/shared/`

**適用範圍**: 跨模組的共用規範、檢查清單、報告等

檔案清單：
```
feature_docs/shared/功能代碼編碼規範.md
feature_docs/shared/功能代碼對應檢查清單.md
feature_docs/shared/mockup_review_report.md
```

### 建立新功能時必須

- [ ] **分配功能代碼**（直接使用功能矩陣編號，如 `[C01]`）
- [ ] **對應功能矩陣編號**（[Cxx] / [Lxx]）
- [ ] **在功能文件中標註功能代碼**（使用標準格式）
- [ ] **在功能矩陣文件中標註實現狀態**（已完成/進行中）
- [ ] **更新本功能代碼總表**

