## 3.7 交易紀錄管理操作

### [L15] 交易查詢

**操作流程**：

```
1. 進入交易列表頁面
2. 設定篩選條件
3. 執行查詢
4. 查看交易詳情
5. 匯出報表
```

**篩選條件**：

| 機台編號 | 文字 | 精確或模糊搜尋 |
| 機台編號 | 文字       | 所屬機台            |
| 交易類型 | 多選       | 儲值/提現/調整/派彩 |
| 狀態     | 多選       | 成功/失敗/處理中    |
| 金額區間 | 數字範圍   | 最小 ~ 最大金額     |

**時間範圍快速篩選**：

- 今日
- 昨日
- 本週
- 上週
- 本月
- 上月
- 自訂區間

### [L18] 交易沖銷

**操作流程**：

```
1. 選取要沖銷的交易
2. 點擊「沖銷」按鈕
3. 填寫沖銷原因
4. 系統驗證沖銷資格
5. 執行沖銷
6. 產生反向交易
7. 記錄沖銷人員
```

**沖銷規則**：

- 僅可沖銷 24 小時內交易
- 需主管授權
- 自動產生沖銷憑證（PDF）

**輸入欄位**：

| 欄位名稱 | 類型     | 必填 | 說明         |
| -------- | -------- | ---- | ------------ |
| 沖銷原因 | 下拉選單 | ✅   | 選擇沖銷原因 |
| 詳細說明 | 文字     | ✅   | 補充說明     |
| 授權碼   | 數字     | ✅   | 主管授權碼   |

### 交易進階操作
- **[L16] 交易詳情檢視**：查詢單筆交易的完整生命週期與資料，包含時間、金額與操作人員。
- **[C44] 交易統計**：提供特定條件下的總結數據，包含總儲值、總派彩與淨利潤。
- **[C45] 交易對帳**：匯入遊戲商的原始帳單進行比對，標示出不一致的問題交易。
- **[C51] 交易導出**：將過濾後的交易紀錄或對帳結果匯出為 CSV 或 Excel 檔案格式。

## 3.14 錢包管理操作

### [C106] 錢包總覽

**功能說明**：
錢包管理提供中央錢包的統一管理，包括餘額查詢、轉帳、對帳等功能。

**操作流程**：

```plaintext
1. 進入「錢包管理」模組
2. 查看中央錢包餘額
3. 檢視各遊戲商錢包餘額
4. 執行轉帳操作
5. 進行對帳
```

**錢包類型**：

| 類型       | 說明                     |
| ---------- | ------------------------ |
| 中央錢包   | 主帳戶，集中管理所有資金 |
| 遊戲商錢包 | 各遊戲供應商帳戶         |

### [C106] 轉帳操作

**操作流程**：

```plaintext
1. 選擇來源錢包
2. 選擇目標錢包
3. 輸入轉帳金額
4. 確認轉帳
5. 記錄日誌
```
