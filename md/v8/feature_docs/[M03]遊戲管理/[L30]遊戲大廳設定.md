### [L30] 遊戲大廳設定

**功能說明**：管理大廳內的遊戲顯示版位、推薦順序與跑馬燈公告等前台呈現要素。

### 遊戲配置與顯示設定
- **[C25] 遊戲配置**：設定遊戲在平台上的各種靜態參數與圖片。
- **[L31] 遊戲參數設定**：調整該遊戲的賠率、難度、最低起押金額等內部參數（依據供應商 API 開放程度而定）。
- **[L32] 遊戲顯示順序**：透過拖曳或設定權重值，調整遊戲在列表或大廳的排列順序。
- **[L33] 遊戲狀態查詢**：即時取得遊戲目前的連線狀況與可用狀態，確認是否在維修中。

### 遊戲測試與模擬下注
- **[C28] 遊戲測試**：提供內部人員在正式發佈前，進行進入遊戲與流程測試之介面。
- **[L43] 虛擬下注**：透過測試帳號進行下注模擬，確認派發結果及扣款邏輯，不影響真實報表。
- **[L44] 遊戲餘額查詢**：在測試或客訴期間，用以查詢使用者與該遊戲連結之錢包餘額。
- **[L45] 遊戲開分/洗分**：提供給測試專用的額度加值與扣除操作，測試充提流程。
- **[L46] 遊戲派彩**：模擬玩家獲勝後，系統將派彩金額寫回該測試錢包的動作。
