## 3.3 多人遊戲操作

### [C57] 牌桌列表檢視與 [C61] 遊戲類型篩選

**功能說明**：
多人遊戲（真人百家樂、龍虎、骰寶等）由遊戲供應商 API 動態提供，牌桌資訊來自第三方系統。此模組包含了 **[C61] 遊戲類型篩選** 功能，幫助快速尋找目標牌桌。

**操作流程**：

```
1. 進入「多人遊戲」模組
2. 查看牌桌列表
3. 使用遊戲類型篩選
4. 點擊牌桌查看詳情
```

**牌桌欄位**：

| 欄位名稱 | 類型   | 說明                |
| -------- | ------ | ------------------- |
| 牌桌編號 | 文字   | 供應商提供的牌桌 ID |
| 遊戲類型 | 標籤   | 百家樂/龍虎/骰寶    |
| 供應商   | 文字   | 所屬遊戲供應商      |
| 狀態     | 標籤   | 開放/關閉/維護中    |
| 最低限注 | 數值   | 最低投注金額        |
| 最高限注 | 數值   | 最高投注金額        |
| 莊家抽水 | 百分比 | 抽水比例            |
| 在線人數 | 數值   | 當前在線機台數      |

### [C58] 牌桌狀態監控

**操作流程**：

```
1. 選擇牌桌
2. 查看即時狀態
3. 監控投注情況
4. 查看異常告警
```

**監控資訊**：

| 監控項目 | 說明                 |
| -------- | -------------------- |
| 當前牌靴 | 牌靴編號、剩餘牌數   |
| 歷史結果 | 最近 20 局結果       |
| 即時投注 | 當前總投注額         |
| 異常狀態 | 連線異常、金額異常等 |

### [C59] 牌桌配置管理

**操作流程**：

```
1. 選擇牌桌
2. 點擊「配置」
3. 設定投注限額與莊家抽水比例
4. 儲存設定
```

**可配置項目** (涵蓋 **[C63] 投注額設定** 與 **[C64] 莊家抽水設定**)：

| 項目     | 類型   | 範圍      | 說明                       |
| -------- | ------ | --------- | -------------------------- |
| 最低限注 | 數值   | 1-999999  | 最低投注金額               |
| 最高限注 | 數值   | 1-9999999 | 最高投注金額               |
| 莊家抽水 | 百分比 | 0-50%     | 莊家贏家抽成               |
| 單注上限 | 數值   | 1-999999  | 單筆投注上限               |
| 單局上限 | 數值   | 1-9999999 | 單局投注上限               |
| 備註     | 文字   | -         | 依照第三方遊戲商的API 調整 |

### [C60] 牌局歷史查詢

**功能說明**：
提供查閱所有多人遊戲歷史牌局紀錄的功能。本模組亦涵蓋：
- **[C66] 荷官操作紀錄**：記錄與監控每局遊戲中的荷官操作。
- **[C67] 牌局歷史匯出**：允許將篩選後的資料匯出為報表。

**操作流程**：

```
1. 進入「牌局歷史」
2. 設定篩選條件
3. 查詢歷史記錄
4. 查看詳情或匯出
```

**篩選條件**：

| 條件     | 類型     | 說明             |
| -------- | -------- | ---------------- |
| 遊戲類型 | 多選     | 百家樂/龍虎/骰寶 |
| 牌桌     | 多選     | 特定牌桌         |
| 時間範圍 | 日期區間 | 查詢區間         |
| 結果     | 多選     | 莊/閒/和         |

### [C62] 牌靴管理

**功能說明**：
管理牌靴（ Shoe ）的副數、張數、換靴規則。

**操作流程**：

```
1. 選擇牌桌
2. 進入「牌靴管理」
3. 設定牌靴參數
4. 查看當前牌靴狀態
```

**設定欄位**：

| 欄位名稱           | 類型     | 說明                       |
| ------------------ | -------- | -------------------------- |
| 牌靴副數           | 數字     | 1-8 副牌                   |
| 剩餘多少張觸發換靴 | 數字     | 建議 20-40 張              |
| 換靴規則           | 下拉選單 | 自動/手動                  |
| 備註               | 文字     | 依照第三方遊戲商的API 調整 |

### [C65] 異常狀態告警

**告警類型**：

| 告警類型 | 說明          | 處理方式            |
| -------- | ------------- | ------------------- |
| 連線中斷 | API 連線異常  | 自動重連/通知管理員 |
| 金額異常 | 投注/派彩異常 | 標記並人工處理      |
| 牌靴異常 | 牌數計算異常  | 強制換靴            |
| 荷官異常 | 荷官操作逾時  | 切換荷官            |
