### [C71] 角色權限設定

**操作流程**：

```
1. 進入「角色管理」
2. 新增/編輯角色
3. 設定功能權限
4. 儲存角色
```

**權限類型**：

| 權限 | 說明           |
| ---- | -------------- |
| 檢視 | 可查看功能畫面 |
| 新增 | 可新增資料     |
| 編輯 | 可修改資料     |
| 刪除 | 可刪除資料     |
| 匯出 | 可匯出資料     |
| 管理 | 可設定權限     |

**預設角色**：

| 角色       | 權限範圍              |
| ---------- | --------------------- |
| 系統管理員 | 全部功能              |
| 站長       | 本站點全部功能        |
| 客服       | 查詢/編輯（不含刪除） |
| 稽核       | 僅檢視/匯出           |

| 權限層級 | 代碼 | 說明             |

#### 3.4.4.3 權限制定原則

```
┌─────────────────────────────────────────────────────┐
│              權限制定流程                             │
├─────────────────────────────────────────────────────┤
│  1. 確認帳號用途                                     │
│     └── 內部人員 → 使用者權限模組                     │
│                                                     │
│  2. 選擇角色                                        │
│     └── 根據工作職責選擇預設角色                      │
│                                                     │
│  3. 客製化調整                                       │
│     └── 依需求增減特定權限                           │
│                                                     │
│  4. 審核確認                                        │
│     └── 主管審核後啟用                               │
└─────────────────────────────────────────────────────┘
```

**核心原則**：

- **最小權限** - 只給完成工作需要的最低權限
- **職責分離** - 稽核與操作分開
- **記錄所有** - 所有權限變更都留日誌

#### 3.4.4.4 動態角色自定義（進階）

系統支援動態角色自定義引擎，可細部勾選 API 權限：

**使用場景**：

- 審計員：僅開放交易查詢 API
- 查帳會計：僅開放報表匯出 API
- 維修外包：僅開放機台狀態查看 API

**自定義權限類型**：

- 功能模組權限
- API 端點權限
- 資料範圍權限（特定站點）
- 時間權限（上班時間才能操作）
