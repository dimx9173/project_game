# [C69] 系統登入

## 📋 概述

本文檔說明集中式後台的**管理員登入系統**，提供安全的帳號密碼認證機制。

---

## 🔐 登入機制

| 項目 | 說明 |
|------|------|
| **認證方式** | 帳號 + 密碼 |
| **權杖類型** | JWT Token |
| **權杖有效期** | 24 小時 |
| **密碼加密** | BCrypt |
| **登入失敗鎖定** | 5 次錯誤後鎖定 30 分鐘 |

## 🔑 功能特點

### 1. 帳號密碼認證
- 支援帳號/Email 登入
- 密碼複雜度檢驗
- 大小寫不敏感

### 2. 記住我
- 延長登入有效期至 7 天
- 使用安全 Token 機制

### 3. 忘記密碼
- 透過 Email 重設密碼
- 重設連結有效期 24 小時

### 4. 登入失敗處理
- 錯誤次數計算
- 帳號鎖定機制
- IP 封鎖保護

---

## 🔄 登入流程

```
使用者輸入帳號密碼
    ↓
系統驗證帳號是否存在
    ↓
驗證密碼（BCrypt 比對）
    ↓
產生 JWT Token
    ↓
回傳 Token 與用戶資訊
    ↓
前端儲存 Token 並跳轉儀表板
```

---

## 📝 API 端點

| 方法 | 路徑 | 說明 |
|------|------|------|
| POST | `/api/auth/login` | 使用者登入 |
| POST | `/api/auth/logout` | 登出 |
| POST | `/api/auth/refresh` | 刷新 Token |
| POST | `/api/auth/forgot-password` | 忘記密碼 |
| POST | `/api/auth/reset-password` | 重設密碼 |
