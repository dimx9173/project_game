### [C59] 牌桌配置管理

**操作流程**：

```
1. 選擇牌桌
2. 點擊「配置」
3. 設定投注限額與莊家抽水比例
4. 儲存設定
```

**可配置項目** (涵蓋 **[C63] 投注額設定** 與 **[C64] 莊家抽水設定**)：

| 項目     | 類型   | 範圍      | 說明                       |
| -------- | ------ | --------- | -------------------------- |
| 最低限注 | 數值   | 1-999999  | 最低投注金額               |
| 最高限注 | 數值   | 1-9999999 | 最高投注金額               |
| 莊家抽水 | 百分比 | 0-50%     | 莊家贏家抽成               |
| 單注上限 | 數值   | 1-999999  | 單筆投注上限               |
| 單局上限 | 數值   | 1-9999999 | 單局投注上限               |
| 備註     | 文字   | -         | 依照第三方遊戲商的API 調整 |
