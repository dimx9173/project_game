## 功能說明

管理代理商的錢包帳戶與資金調動。

### 功能特性
- 代理商錢包餘額查詢
- 資金轉帳記錄
- 錢包歸集設定
- 轉帳限額設定

### 使用場景
- 代理商餘額管理
- 收益分配
- 資金調動審批

---

## API Endpoint

`GET /api/agents/wallets`
`POST /api/agents/wallets/transfer`
