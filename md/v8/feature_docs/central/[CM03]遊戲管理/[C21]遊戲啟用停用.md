# [C21] 遊戲啟用/停用

**功能代碼**: FLG-CEN-C21  
**功能矩陣對應**: [C21] 遊戲啟用/停用  
**模組**: [M03]遊戲管理  
**版本**: 旗艦版  

---

## 功能說明

集中式後台可遠端控制遊戲的啟用與停用狀態。

### 功能特性
- 單一遊戲啟用/停用
- 批量遊戲啟用/停用
- 定時排程啟用/停用
- 記錄操作日誌

### 使用場景
- 遊戲維護時停用
- 活動結束後停用
- 新遊戲上線時啟用

---

## API Endpoint

`POST /api/games/{id}/status`
