### [C78] 前端大廳配置 (/ui-config)

**功能說明**：
提供管理大廳前端 UI 佈景（與顯示比例）的預設功能，包含主題顏色、背景圖示及風格選項。機台重新載入後即可自動從此端點抓取最新配置。
| 建立時間 | 日期區間 | 限定機台新增時間 |
| 餘額範圍 | 數值區間 | 限定機台餘額範圍 |
| 機台類型 | 多選     | 限定特定機台     |
| 遊戲類型 | 多選     | 限定特定遊戲     |

**預設目標選項**：

| 選項     | 說明                 |
| -------- | -------------------- |
| 全體機台 | 發送給所有機台       |
| 在線機台 | 僅發送給當前在線機台 |
| 指定機台 | 選擇特定機台發送     |
| 指定站點 | 選擇特定站點發送     |

#### 3.5.4.3 排程廣播

**排程類型**：

| 類型     | 說明               | 範例              |
| -------- | ------------------ | ----------------- |
| 立即發送 | 立即推送到客戶端   | 緊急公告          |
| 延遲發送 | 設定延遲時間後發送 | 5分鐘後提醒       |
| 定時發送 | 指定精確時間發送   | 每日下午3點       |
| 循環發送 | 設定循環週期發送   | 每小時提醒        |
| 條件觸發 | 滿足條件自動發送   | 餘額低於100元提醒 |

**排程參數**：

| 參數     | 類型     | 說明                 |
| -------- | -------- | -------------------- |
| 開始時間 | 日期時間 | 排程開始時間         |
| 結束時間 | 日期時間 | 排程結束時間         |
| 發送間隔 | 數字     | 循環發送間隔（分鐘） |
| 發送次數 | 數字     | 循環發送次數上限     |
| 時區     | 下拉選單 | 時區設定             |

#### 3.5.4.4 廣播效果追蹤

**追蹤指標**：

| 指標   | 說明               | 計算方式              |
| ------ | ------------------ | --------------------- |
| 送達率 | 訊息成功送達比例   | 成功送達數 / 發送總數 |
| 閱讀率 | 機台端閱讀訊息比例 | 閱讀數 / 送達數       |
| 點擊率 | 機台端點擊連結比例 | 點擊數 / 閱讀數       |
| 忽略率 | 機台端未閱讀比例   | 未閱讀數 / 送達數     |

**效果報表欄位**：

| 廣播編號 | 文字     | 系統自動產生 |
| 廣播標題 | 文字     | 廣播訊息標題 |
| 類型     | 標籤     | 廣播類型     |
| 發送時間 | 日期時間 | 實際發送時間 |
| 目標數   | 數字     | 目標機台數   |
| 送達數   | 數字     | 成功送達數   |
| 閱讀數   | 數字     | 已閱讀數     |
| 點擊數   | 數字     | 點擊連結數   |

**匯出功能**：

- 支援匯出效果報表為 CSV 或 Excel
- 支援篩選特定時間範圍

#### 3.5.4.5 輸入欄位總覽

| 廣播標題 | 文字     | ✅   | 廣播訊息標題          |
| 廣播類型 | 下拉選單 | ✅   | 選擇廣播類型          |
| 廣播內容 | 文字     | ✅   | 訊息內容（支援 HTML） |
| 目標範圍 | 多選     | ✅   | 選擇目標機台          |
| 篩選條件 | 複合     | -    | 進階篩選條件          |
| 發送時間 | 日期時間 | ✅   | 即時/排程選擇         |
| 顯示時長 | 數字     | -    | 訊息顯示秒數          |
| 排序權重 | 數字     | -    | 多則訊息排序優先級    |
