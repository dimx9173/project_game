### [C76] 公告管理

**操作流程**：

```
1. 進入「公告管理」
2. 新增/編輯公告
3. 設定發布時間
4. 選擇顯示位置
5. 儲存公告
```

**公告欄位**：

| 欄位名稱 | 類型     | 說明                  |
| -------- | -------- | --------------------- |
| 標題     | 文字     | 公告標題              |
| 內容     | 文字     | 公告內容（支援 HTML） |
| 類型     | 下拉選單 | 系統/活動/維護        |
| 顯示位置 | 多選     | 大廳/遊戲內/登入頁    |
| 開始時間 | 日期時間 | 顯示開始時間          |
| 結束時間 | 日期時間 | 顯示結束時間          |
| 狀態     | 開關     | 啟用/停用             |
