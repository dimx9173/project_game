### [C104] 餘額查詢與轉帳

**操作流程**：

```
1. 選擇遊戲商
2. 查詢當前餘額
3. 執行轉入/轉出
4. 確認交易
5. 記錄日誌
```

**轉帳類型**：

| 類型 | 說明                 |
| ---- | -------------------- |
| 轉入 | 從中央錢包轉入遊戲商 |
| 轉出 | 從遊戲商轉回中央錢包 |
| 對帳 | 同步餘額資料         |
