### [C01] 儀表板總覽

**功能說明**：
儀表板是系統首頁，提供全平台 KPI 監控、營收趨勢、機台狀態等即時資訊。

**操作流程**：

```
1. 登入系統後自動進入儀表板
2. 檢視各項 KPI 指標卡片
3. 查看趨勢圖與統計圖表
4. 點擊異常項目進入詳細頁面
```

**儀表板元件**：

| 元件名稱                 | 類型     | 說明                       |
| ------------------------ | -------- | -------------------------- |
| KPI 卡片                 | 數值顯示 | 顯示關鍵指標數值與變化趨勢 |
| [L01] 營收趨勢圖         | 折線圖   | 顯示近7天/30天營收趨勢     |
| [L02] 機台狀態圖         | 圓餅圖   | 顯示線上/離線/異常機台比例 |
| [L03] 熱門遊戲排行       | 排行榜   | 顯示前10名熱門遊戲         |
| [C05] 系統通知           | 列表     | 顯示最新系統訊息與告警     |
| [L04] 今日營收/下注/派彩 | 數值     | 顯示今日營收概況           |
| [C02] 硬體/網路狀態      | 狀態列   | 顯示本機連線與硬體健康狀態 |

**篩選功能**：

| 篩選條件 | 類型     | 說明                        |
| -------- | -------- | --------------------------- |
| 時間範圍 | 下拉選單 | 今日/昨日/近7天/近30天/自訂 |
| 站點選擇 | 多選     | 選擇特定站點查看            |
| 遊戲類型 | 多選     | 篩選特定遊戲類型            |

**匯出功能**：

- 支援匯出儀表板數據為 CSV 或 Excel 格式
- 可選擇匯出特定時間範圍的數據
