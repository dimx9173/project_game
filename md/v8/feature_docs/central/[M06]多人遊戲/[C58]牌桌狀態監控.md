### [C58] 牌桌狀態監控

**操作流程**：

```
1. 選擇牌桌
2. 查看即時狀態
3. 監控投注情況
4. 查看異常告警
```

**監控資訊**：

| 監控項目 | 說明                 |
| -------- | -------------------- |
| 當前牌靴 | 牌靴編號、剩餘牌數   |
| 歷史結果 | 最近 20 局結果       |
| 即時投注 | 當前總投注額         |
| 異常狀態 | 連線異常、金額異常等 |
