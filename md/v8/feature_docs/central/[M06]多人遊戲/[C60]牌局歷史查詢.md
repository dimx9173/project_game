### [C60] 牌局歷史查詢

**功能說明**：
提供查閱所有多人遊戲歷史牌局紀錄的功能。本模組亦涵蓋：
- **[C66] 荷官操作紀錄**：記錄與監控每局遊戲中的荷官操作。
- **[C67] 牌局歷史匯出**：允許將篩選後的資料匯出為報表。

**操作流程**：

```
1. 進入「牌局歷史」
2. 設定篩選條件
3. 查詢歷史記錄
4. 查看詳情或匯出
```

**篩選條件**：

| 條件     | 類型     | 說明             |
| -------- | -------- | ---------------- |
| 遊戲類型 | 多選     | 百家樂/龍虎/骰寶 |
| 牌桌     | 多選     | 特定牌桌         |
| 時間範圍 | 日期區間 | 查詢區間         |
| 結果     | 多選     | 莊/閒/和         |
