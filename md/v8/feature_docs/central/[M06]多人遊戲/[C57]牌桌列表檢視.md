### [C57] 牌桌列表檢視與 [C61] 遊戲類型篩選

**功能說明**：
多人遊戲（真人百家樂、龍虎、骰寶等）由遊戲供應商 API 動態提供，牌桌資訊來自第三方系統。此模組包含了 **[C61] 遊戲類型篩選** 功能，幫助快速尋找目標牌桌。

**操作流程**：

```
1. 進入「多人遊戲」模組
2. 查看牌桌列表
3. 使用遊戲類型篩選
4. 點擊牌桌查看詳情
```

**牌桌欄位**：

| 欄位名稱 | 類型   | 說明                |
| -------- | ------ | ------------------- |
| 牌桌編號 | 文字   | 供應商提供的牌桌 ID |
| 遊戲類型 | 標籤   | 百家樂/龍虎/骰寶    |
| 供應商   | 文字   | 所屬遊戲供應商      |
| 狀態     | 標籤   | 開放/關閉/維護中    |
| 最低限注 | 數值   | 最低投注金額        |
| 最高限注 | 數值   | 最高投注金額        |
| 莊家抽水 | 百分比 | 抽水比例            |
| 在線人數 | 數值   | 當前在線機台數      |
