## 操作流程

### 2.1 自動對帳流程

```
┌─────────────────────────────────────────────────────────────┐
│  1. 系統定時觸發                                            │
│     - 每日 03:00 自動執行                                    │
│     - 也可手動觸發                                          │
│     ↓                                                       │
│  2. 資料蒐集                                                │
│     - 取得 Central 交易記錄                                 │
│     - 取得 Local 交易記錄                                   │
│     ↓                                                       │
│  3. 比對計算                                                │
│     - 逐筆比對交易                                          │
│     - 計算差異金額                                          │
│     ↓                                                       │
│  4. 產生結果                                                │
│     - 對帳成功：差異 = 0                                    │
│     - 對帳異常：差異 ≠ 0                                    │
│     ↓                                                       │
│  5. 記錄日誌                                                │
│     - 記錄對帳時間、內容、結果                               │
│     - 產生對帳報告                                          │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 對帳項目

| 對帳項目 | 說明 | 容許差異 |
|----------|------|----------|
| **餘額對帳** | 中央餘額 vs 本地餘額 | ≤ 0.01 元 |
| **交易筆數** | 交易記錄筆數比對 | 0 筆 |
| **開分總額** | 開分金額合計比對 | ≤ 0.01 元 |
| **洗分總額** | 洗分金額合計比對 | ≤ 0.01 元 |
| **遊戲輸贏** | 遊戲結果比對 | ≤ 0.01 元 |

### 2.3 查看對帳結果

```
┌─────────────────────────────────────────────────────────────┐
│  1. 進入「交易紀錄管理」->「交易對帳」                       │
│     ↓                                                       │
│  2. 選擇對帳時間                                            │
│     - 選擇特定日期                                          │
│     - 或選擇機台                                            │
│     ↓                                                       │
│  3. 查看對帳狀態                                            │
│     - 綠色：對帳成功                                        │
│     - 黃色：對帳有差異                                      │
│     - 紅色：對帳失敗                                        │
│     ↓                                                       │
│  4. 查看詳情                                                │
│     - 點擊查看每筆交易比對結果                              │
│     - 顯示差異金額與原因                                    │
└─────────────────────────────────────────────────────────────┘
```

### 2.4 差異處理流程

```
┌─────────────────────────────────────────────────────────────┐
│  1. 發現差異                                                 │
│     - 系統標記為「待處理」                                   │
│     - 發送通知給管理員                                      │
│     ↓                                                       │
│  2. 分析差異原因                                            │
│     - 調閱原始交易記錄                                      │
│     - 比對時間戳                                            │
│     - 檢查是否有遺漏或重複                                   │
│     ↓                                                       │
│  3. 處理方式                                                │
│     - 中央調整：以中央記錄為準                               │
│     - 本地調整：以本地記錄為準                               │
│     - 人工審核：無法自動判定                                │
│     ↓                                                       │
│  4. 記錄處理結果                                            │
│     - 記錄調整原因                                          │
│     - 記錄處理人員                                          │
│     - 重新對帳確認                                          │
└─────────────────────────────────────────────────────────────┘
```

### 2.5 異常對帳類型

| 異常類型 | 說明 | 處理方式 |
|----------|------|----------|
| **餘額不平** | 中央餘額 ≠ 本地餘額 | 差異 < 0.01 自動調整，否則人工審核 |
| **交易遺漏** | 中央有記錄但本地沒有 | 補錄本地交易或標記異常 |
| **交易重複** | 同一筆交易出現兩次 | 標記並沖銷重複項目 |
| **金額差異** | 同一筆交易金額不同 | 以時間戳較早者為準 |
| **時間差異** | 交易時間對不上 | 檢查時區與同步延遲 |

### 2.6 手動對帳

```
┌─────────────────────────────────────────────────────────────┐
│  1. 點擊「手動對帳」                                        │
│     ↓                                                       │
│  2. 選擇對帳範圍                                            │
│     - 全部機台 / 指定機台                                   │
│     - 全部時間 / 指定時間範圍                               │
│     ↓                                                       │
│  3. 執行對帳                                                │
│     - 系統立即執行比對                                      │
│     - 顯示進度                                              │
│     ↓                                                       │
│  4. 查看結果                                                │
│     - 顯示對帳成功/失敗                                     │
│     - 顯示差異清單                                          │
└─────────────────────────────────────────────────────────────┘
```

### 2.7 對帳日誌

- **記錄內容**：對帳時間、執行人員、對帳範圍、結果統計、差異明細
- **保留期限**：預設保留 365 天
- **匯出功能**：可匯出 PDF/Excel 用於稽核
