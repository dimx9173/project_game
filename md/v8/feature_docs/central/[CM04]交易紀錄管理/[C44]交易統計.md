## 操作流程

### 2.1 查看交易統計

```
┌─────────────────────────────────────────────────────────────┐
│  1. 進入「交易紀錄管理」->「交易統計」                       │
│     ↓                                                       │
│  2. 選擇時間範圍                                            │
│     - 今日/昨日/本週/本月/自訂日期                           │
│     ↓                                                       │
│  3. 選擇篩選維度 (可多選)                                   │
│     - 門市：依分店/門市篩選                                │
│     - 機台：依機台編號/名稱篩選                            │
│     - 遊戲：依遊戲類型篩選                                 │
│     - 遊戲商：依供應商篩選                                  │
│     ↓                                                       │
│  4. 查看統計結果                                            │
│     - 總營收 = 派彩 - 下注                                 │
│     - 下注總額                                              │
│     - 派彩總額                                              │
│     - 遊戲局數                                              │
│     - 營收 RTP                                              │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 統計維度與篩選

| 維度 | 說明 | 範例 |
|------|------|------|
| **門市** | 按分店/門市篩選 (可多選) | 台北門市A區、台中門市、高雄門市 |
| **機台** | 按機台篩選 (可多選) | 機台編號、機台名稱 |
| **遊戲** | 按遊戲類型篩選 (可多選) | 老虎機、捕魚機、棋牌 |
| **遊戲商** | 按遊戲供應商篩選 (可多選) | RG電子、JDB電子、RT電子 |
| **交易類型** | 開分/洗分/全部 | 篩選特定交易類型 |

### 2.3 統計圖表展示

```
┌─────────────────────────────────────────────────────────────┐
│  1. 趨勢圖                                                  │
│     - 折線圖：顯示營收趨勢                                  │
│     - 柱狀圖：比較不同時段營收                              │
│     ↓                                                       │
│  2. 分布圖                                                  │
│     - 圓餅圖：各遊戲/機台營收占比                          │
│     - 條形圖：Top 10 機台/遊戲                            │
│     ↓                                                       │
│  3. 比較圖                                                  │
│     - 同期比較：與昨日/上週/上月同期對比                    │
│     - 目標比較：與設定目標營收對比                          │
└─────────────────────────────────────────────────────────────┘
```

### 2.4 統計欄位說明

| 欄位 | 說明 | 計算方式 |
|------|------|----------|
| **總營收** | 總派彩 - 總下注 | 派彩總額 - 下注總額 |
| **下注總額** | 所有下注金額合計 | SUM(bet_amount) |
| **派彩總額** | 所有派彩金額合計 | SUM(payout_amount) |
| **遊戲局數** | 總遊戲次數 | COUNT(game_sessions) |
| **營收 RTP** | 派彩率 | 派彩總額 / 下注總額 × 100% |
| **平均客单价** | 每筆交易平均金額 | 總營收 / 交易筆數 |
| **上線機台數** | 有交易的機台數 | COUNT(DISTINCT machine_id) |

### 2.5 匯出功能

```
┌─────────────────────────────────────────────────────────────┐
│  1. 設定篩選條件                                            │
│     - 選擇時間範圍                                           │
│     - 選擇維度                                               │
│     ↓                                                       │
│  2. 點擊「匯出」                                            │
│     ↓                                                       │
│  3. 選擇格式                                                │
│     - CSV：通用格式                                         │
│     - Excel (.xlsx)：含格式                                 │
│     - PDF：報表格式                                         │
│     ↓                                                       │
│  4. 選擇匯出內容                                            │
│     - 僅統計摘要                                           │
│     - 包含明細                                              │
│     ↓                                                       │
│  5. 下載檔案                                                │
│     - 系統產生下載連結                                       │
│     - 郵件寄送 (可選)                                       │
└─────────────────────────────────────────────────────────────┘
```

### 2.6 數據來源

- **即時資料**：來自 Central 資料庫的交易記錄表
- **同步延遲**：Local 交易資料每分鐘同步到 Central
- **統計粒度**：支持分鐘/小時/日/週/月級別統計
