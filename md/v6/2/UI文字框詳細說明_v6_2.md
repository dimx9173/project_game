## 三、通用元件說明

### 3.1 按鈕樣式

| 按鈕類型 | 範例 | 使用時機 |
|----------|------|----------|
| 主要按鈕 | [ 登 入 系 統 ] | 重要動作 |
| 次要按鈕 | [ 取 消 ] | 取消動作 |
| 危險按鈕 | [ 刪 除 ] | 不可復原操作 |
| 圖標按鈕 | [✏️] [👁] [🗑️] | 列表操作 |

### 3.2 狀態顏色

| 狀態 | 顏色 | 範例 |
|------|------|------|
| 成功/正常 | 綠色 | 🟢 正常、✅ 成功 |
| 警告/注意 | 黃色 | 🟡 維護、⏳ 處理中 |
| 錯誤/離線 | 紅色 | 🔴 離線、❌ 失敗 |
| 資訊 | 藍色 | ℹ️ 說明、📢 公告 |

> **備註**：以上為 v6/2 基礎商用版的 UI 文字框說明。v6/3 和 v6/4 版本會在此基礎上增加更多功能頁面的詳細說明。
