# 遊戲平台管理系統 - 工時評估總表

## 版本選項

| 版本 | 開發時程 | 人天 | 預算 | 適用場景 |
|------|----------|------|------|----------|
| **MVP（2個月）** | 2 個月 | 50 人天 | 40 萬 | 快速落地、快速驗證 |
| **標準版（3個月）** | 3 個月 | 80 人天 | 60 萬 | 一般營運、代理商體系 |
| **完整版（4個月）** | 4 個月 | 120 人天 | 80 萬 | 百台規模、維運自動化 |

---

## MVP 版（2個月 / 50 人天）

### 工時分配

| 項目分類 | 內容重點 | 人天 | 佔比 |
| :--- | :--- | :---: | :---: |
| **系統基建與單機 Queue** | PostgreSQL + SQLite + 斷線重試佇列 | 12 | 24% |
| **實用型集中大後台** | 兩層實體驗證、現代化過濾大表單、無圖表 | 20 | 40% |
| **單機本地後/前端** | 視覺化設定檔介面、現場 PIN 碼打分功能 | 18 | 36% |
| **合計** | | **50** | **100%** |

### 功能重點

- ✅ 雙層權限（超級管理員 + 操作員）
- ✅ 斷線重試佇列（資料不遺失）
- ✅ 現代化 DataGrid 列表（分頁、篩選、排序）
- ✅ 本機 UI 設定（取代 .env）
- ✅ PIN 碼保護開洗分
- ❌ 無圖表視覺化
- ❌ 無第三方遊戲串接
- ❌ 無心跳監控

---

## 標準版（3個月 / 80 人天）

### 工時分配

| 項目分類 | 內容重點 | 人天 | 佔比 |
| :--- | :--- | :---: | :---: |
| **基礎建設** | MVP 階段工時 | 40 | 50% |
| **三層式權限與代理商模組** | 使用者表隔離、代理商樹狀結構 | 15 | 19% |
| **中央視覺化與心跳監控** | 圖表 Dashboard、機台紅綠燈清單 | 10 | 12% |
| **外部遊戲商串接** | 第三方 API 錢包互打、超時邏輯防護 | 15 | 19% |
| **合計** | | **80** | **100%** |

### 功能重點

- ✅ 包含 MVP 所有功能
- ✅ 三層權限（超級管理員 + 代理商 + 操作員）
- ✅ 視覺化 Dashboard（營收趨勢圖、機台狀態圓餅圖）
- ✅ Heartbeat 心跳監控（1分鐘間隔）
- ✅ 第三方遊戲 API 串接（1-3 家）
- ✅ 中心錢包（轉入/轉出）
- ❌ 無 OTA 遠端更新
- ❌ 無進階硬體監控

## 完整版（4個月 / 120 人天）

### 工時分配

| 項目分類 | 內容重點 | 人天 | 佔比 |
| :--- | :--- | :---: | :---: |
| **前三階段基礎** | MVP + 標準版工時 | 80 | 67% |
| **遠端營運 OTA 模組** | 後台套件上傳、單機下載防護重啟邏輯 | 20 | 17% |
| **進階硬體監控回報** | CPU/RAM 取樣回傳、後台面板呈現 | 8 | 6% |
| **對帳與會計客訴專區** | 爭議紀錄沖銷、失敗隊列人工介入 UI | 12 | 10% |
| **合計** | | **120** | **100%** |

### 功能重點

- ✅ 包含標準版所有功能
- ✅ OTA 遠端版本派送（自動下載、MD5 驗證、熱更新）
- ✅ 進階硬體監控（CPU、RAM、網路延遲）
- ✅ 設備健康度儀表板
- ✅ 錯帳與爭議帳單管理
- ✅ 沖銷憑證產生
- ✅ 符合正規金流會計核銷邏輯

## 遊戲串接時程

| 遊戲商數量 | 額外工時 | 說明 |
|------------|----------|------|
| **第一家** | 2 週（10 人天） | 建立串接框架 + 熟悉流程 |
| **同類型第二家起** | +1 週/家 | 差異化適配 |
| **5 家以內** | 3-4 週 | 標準報價 |
| **6-10 家** | +3-4 週 | 視遊戲商類型差異 |

### 影響時程變數

1. 遊戲商 API 文件完整性
2. 遊戲商技術支援响应速度
3. 特殊功能需求（風控、彩池等）
4. 測試環境是否由遊戲商提供
5. 是否為同類型遊戲
