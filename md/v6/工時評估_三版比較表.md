# 工時評估 - 三版本比較表（包含五家內遊戲串接）

> **備註**：本版本為經過 SA 分析後的修正版本，加入了**5 家遊戲 API 串接**、測試時間與風險 Buffer。
> 以 **1 名資深全端工程師** 獨立開發為基準。人天單價以 8,000 TWD/天 估算。

---

## 快速比較

| 項目 | MVP | 標準版 | 旗艦版 |
|------|-----|--------|--------|
| **開發時程** | 2.5 個月 (約 10 週) | 3 個月 (約 12.5 週) | 4 個月 (約 17 週) |
| **預估人天（純開發）** | 38 人天 | 48 人天 | 66 人天 |
| **測試與調試** | +10 人天 | +14 人天 | +19 人天 |
| **總人天（含測試）** | **48 人天** | **62 人天** | **85 人天** |
| **基本預算** | 38.4 萬 | 49.6 萬 | 68.0 萬 |
| **建議預算（含 15% Buffer）** | **44.2 萬** | **57.1 萬** | **78.2 萬** |

## 風險評估與建議

### 風險矩陣

| 風險項目 | 發生機率 | 影響程度 | 緩解措施 |
|---------|---------|---------|---------|
| 第三方 API 規格變動 | 高 | 高 | 先做 Mock Server 隔離依賴 |
| 單人開發請假/生病 | 中 | 高 | 預留 10% 時間 buffer |
| 技術難題卡關 | 中 | 中 | 預留 5% 技術攻堅時間 |
| 需求變更 | 低 | 高 | 採用敏捷迭代，每週 review 範圍 |

### 建議開發節奏

| 週次 | MVP 內容 | 標準版內容 | 旗艦版內容 |
|------|---------|-----------|-----------|
| 1-3 | 系統基建、Queue 機制、第三方 API 串接 | - | - |
| 4-6 | 集中式/單機本地 後端與前端 | - | - |
| 7-10 | 測試、調整、部署 | MVP + 測試 | - |
| 11-12 | - | Dashboard、權限模組、測試部署 | 標準版 |
| 13-15 | - | - | OTA 熱更新、爭議對帳 |
| 16-17 | - | - | 硬體監控、測試部署 |
