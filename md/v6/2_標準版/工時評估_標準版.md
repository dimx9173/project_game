# 遊戲平台管理系統工時評估 - 標準版

## 1. 開發時程

**開發時程：標準版（約 70 工作天/Man-Days，約 3.5 個月）**  
**預算規模：64.4 萬**

## 2. 功能重點

本版是兼顧「系統穩定度」與「代理商營運需求」的**黃金比例版本**。在基礎商用版的之上，加入了**三層式權限架構**以及對**外部遊戲 API** 的支援。

> **📌 遊戲串接**：本版本預設包含 **3 家**第三方遊戲串接（與 MVP 版相同）。

## 3. 詳細功能規格與系統升級點

### 3.1 增加三層營運權限架構（預估工時：15 MD）

- **RBAC 代理商隔離**：
  - 在「超級管理員」與「操作員」之間，加入「代理商 (Agent)」階層。
  - 代理商只能看到自己名下的機台營收、機台狀態。超級管理員則能俯視全域。

### 3.2 Dashboard 與機台心跳監控（預估工時：10 MD）

- **視覺化 Dashboard**：
  - 中央後台加入首頁戰情室。具備昨日/今日營收長條圖，以及活躍機台圓餅圖。
- **Heartbeat 即時監控**：
  - 機台端背景每 1 分鐘發送 Ping 至中央。
  - 中央建立「機台狀態一覽表」，顯示該機台是 🟢在線 還是 🔴斷線，讓管理員能第一時間掌握設備異常。

### 3.3 外部第三方遊戲接入能力（預估工時：15 MD）

- **API 抽象層 (Wallet)**：
  - 新增玩家在機台端的「中心錢包」概念。
  - 支援介接最多 3 家熱門的第三方遊戲 API (如: 打魚機、百家樂)。
  - 提供標準的 Deposit (轉入) / Withdraw (轉出) 機制，玩家在螢幕上點選第三方遊戲時，系統會自動在背景將點數扣至該廠商額度，退出時再補回。
